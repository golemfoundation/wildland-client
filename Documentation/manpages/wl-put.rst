:command:`wl put` - put a file into wildland
============================================

Synopsis
--------

| :command:`wl put [<local-path>] <wildland-path>`

Description
-----------

Put a |~| file into wildland, under given path. The filesystem does not need to
be mounted.

If local path is not given, read standard input.

Options
-------

No options.
